the Crystal Goblet

In this typesetting exercise, an excerpt from "The Crystal Goblet" by Beatrice Warde is set in a traditional formatting style, alluding to the main argument presented in the essay: good type setting should be invisible to the reader like a crystal goblet.

For my take on what invisible typesetting should look like, I opted for the serif typeface Georgia, using a smaller point size and larger margins to create a elegant, classical reading experience.
