myTurtle

For my final project in my Object-Orientated Programming class, I created a game using Java and JavaFx. In myTurtle, the player can by turtles and furniture, then place them on the board.

The board utilizes a 2D array to organize the turtles and furniture items on it. A breadth-first-search is used to calculate the paths of the turtles from the shop to their selected sqaures.
